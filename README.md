#  Go Weather Application

A simple command-line Weather Application built using Go that fetches real-time weather data from an external API.

# Features
- Fetches live weather by city name
- Displays temperature, humidity, and conditions
- Uses Go's net/http and JSON decoding

# Tech Stack
- Go (Golang)
- OpenWeatherMap API
- REST APIs
- JSON

# How to Run
1. Clone the repository
2. Add your API key inside `main.go`
3. Run:
   ```bash
   go run main.go
